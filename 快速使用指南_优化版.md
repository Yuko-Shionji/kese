## 🛠️ 使用新工具类

### UIStyles 示例
```java
// 创建主要按钮
JButton btn = UIStyles.createPrimaryButton("提交");

// 创建输入框
JTextField field = UIStyles.createTextField(20);

// 创建卡片面板
JPanel card = UIStyles.createCardPanel(UIStyles.PRIMARY_COLOR);
```

### NotificationUtil 示例
```java
// 成功提示
NotificationUtil.showSuccess(this, "保存成功！");

// 错误提示
NotificationUtil.showError(this, "操作失败！");

// 确认对话框
boolean confirmed = NotificationUtil.showConfirm(this, "确定要删除吗？");
```

## ⚙️ 可选依赖

### Apache POI（用于 Excel 导出）
如需使用数据导出功能，需要添加 Apache POI 库：

**Maven:**
```xml
<dependency>
    <groupId>org.apache.poi</groupId>
    <artifactId>poi-ooxml</artifactId>
    <version>5.2.3</version>
</dependency>
```

**手动添加:**
1. 下载 `poi-5.2.3.jar` 和 `poi-ooxml-5.2.3.jar`
2. 放入 `lib/` 目录
3. 重新编译即可

## 📞 常见问题

### Q: 界面显示乱码
A: 确保系统安装了"微软雅黑"字体，或修改 UIStyles.java 中的字体设置。

### Q: 编译时找不到符号
A: 检查 classpath 是否正确设置，确保包含了所有依赖的 jar 包。

### Q: 导出功能无法使用
A: 导出功能需要 Apache POI 依赖，目前显示的是提示信息。

### Q: 图表不显示
A: 确保数据加载成功，图表使用模拟数据演示，实际使用需连接真实数据库。
