# 快速使用指南

## 一、环境准备（10 分钟）

### 1. 安装 JDK
- 下载 JDK 8 或更高版本
- 安装并配置环境变量 JAVA_HOME

### 2. 安装 MySQL
推荐使用免安装版：
- 下载 MySQL 5.5+ 免安装版
- 解压到任意目录
- 初始化数据库
- 启动 MySQL 服务

### 3. 下载 MySQL 驱动
- mysql-connector-java-5.1.47.jar
- 放入项目 lib 目录或添加到 classpath

## 二、数据库初始化（5 分钟）

```sql
-- 1. 登录 MySQL
mysql -u root -p

-- 2. 执行脚本
source E:\kese1\database\schema.sql

-- 3. 验证
USE rural_water_db;
SHOW TABLES;  -- 应该看到 10 张表
SELECT * FROM users;  -- 应该看到 4 个测试用户
```

## 三、修改配置（2 分钟）

编辑 `src/com/ruralwater/util/DBUtil.java`:

```java
private static final String URL = "jdbc:mysql://localhost:3306/rural_water_db?useSSL=false&characterEncoding=utf8";
private static final String USERNAME = "root";  // 你的 MySQL 用户名
private static final String PASSWORD = "your_password";  // 你的 MySQL 密码
```

## 四、运行项目

### 方法 1：IDE 运行（推荐）
1. 打开 IntelliJ IDEA 或 Eclipse
2. 导入项目：File -> Open -> 选择 E:\kese1
3. 配置 MySQL 驱动依赖
4. 右键运行 `MainApp.java`

### 方法 2：命令行运行
```bash
cd E:\kese1
javac -d bin src\com\ruralwater\**\*.java
java -cp "bin;lib/mysql-connector-java-5.1.47.jar" com.ruralwater.MainApp
```

## 八、常见问题解决

### Q1: 驱动加载失败
A: 确保 mysql-connector-java jar 包在 classpath 中

### Q2: 无法连接数据库
A: 
- 检查 MySQL 服务是否启动
- 检查用户名密码是否正确
- 检查数据库是否存在

### Q3: 中文乱码
A: 
- 确保数据库字符集为 utf8
- 连接字符串包含 characterEncoding=utf8

### Q4: 编译错误
A: 
- 确保 JDK 版本 1.8+
- 确保源文件编码为 UTF-8
