## 三、技术特性

### 1. 面向对象设计
#### 封装性
- 8 个 JavaBean 实体类（User, Region, WaterSource, WaterPlant, WaterQualityStandard, 
  WaterQualityRecord, WaterQualityDetail, Warning）
- 所有属性私有化，提供公共 getter/setter
- 实现 Serializable 接口支持序列化

#### 继承性
- BaseDAO<T> 泛型接口定义基础 CRUD 方法
- 各 DAO 实现类继承 BaseDAO 接口
- 代码复用率高，结构清晰

#### 多态性
- UserDAO, WaterPlantDAO, WaterQualityRecordDAO 实现同一接口
- 运行时动态绑定具体实现
- 便于扩展和维护

### 2. JDBC 应用
#### 原生 JDBC 编程
- DBUtil 工具类统一管理数据库连接
- PreparedStatement 防止 SQL 注入
- 手动管理事务（beginTransaction, commit, rollback）
- 资源关闭（Connection, Statement, ResultSet）

#### 典型应用
```java
// 防 SQL 注入
PreparedStatement pstmt = conn.prepareStatement(sql);
pstmt.setString(1, username);
pstmt.setString(2, password);

// 事务处理
DBUtil.beginTransaction(conn);
// 多个 SQL 操作
DBUtil.commit(conn);
// 异常时回滚
DBUtil.rollback(conn);
```
