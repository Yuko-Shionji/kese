# 农村饮水安全监测管理系统

## 项目简介
基于 JavaSE Swing + 原生 JDBC + MySQL 开发的农村饮水安全监测管理系统，符合课程设计所有要求。

## 技术栈
- **开发语言**：Java (JDK 8+)
- **GUI 框架**：Java Swing
- **数据库**：MySQL 5.5+
- **数据库连接**：原生 JDBC（无 ORM 框架）
- **架构**：C/S 结构桌面应用
- **版本控制**：Git

## 核心功能
### 1. 用户登录模块
- 用户身份验证
- 权限控制（管理员、操作员、查看员）
- 密码修改

### 2. 水厂管理模块
- 水厂信息 CRUD 操作
- 关键字搜索、分页显示
- 跨表关联查询

### 3. 水质检测模块
- 检测记录管理
- 多条件组合查询
- 审核流程（带事务处理）

### 4. 预警信息模块
- 自动预警（数据库触发器）
- 预警级别分类
- 状态跟踪

### 5. 系统管理模块
- 用户管理、区域管理
- 水源管理、检测标准管理
- 数据初始化（存储过程）

## 快速开始

### 1. 环境准备
- JDK 8+
- MySQL 5.5+
- MySQL 驱动：mysql-connector-java-5.1.x.jar

### 2. 数据库初始化
```bash
mysql -u root -p < database/schema.sql
```

### 3. 修改配置
编辑 `src/com/ruralwater/util/DBUtil.java`，修改数据库连接信息。

### 4. 运行项目
```bash
# IDE 运行
右键运行 MainApp.java

# 或命令行
javac -d bin src\com\ruralwater\**\*.java
java -cp "bin;lib/mysql-connector-java-5.1.47.jar" com.ruralwater.MainApp
```
