**新增功能：**

**代码示例：**
```java
// 事务处理示例
public void addRecord(WaterQualityRecord record, List<WaterQualityDetail> details) {
    Connection conn = null;
    try {
        conn = DBUtil.getConnection();
        DBUtil.beginTransaction(conn);
        
        // 插入主记录
        recordDAO.insertWithConnection(record, conn);
        
        // 获取生成的记录 ID
        Integer recordId = getLastInsertId(conn);
        
        // 为详情设置记录 ID 并批量插入
        for (WaterQualityDetail detail : details) {
            detail.setRecordId(recordId);
        }
        detailService.addDetails(details);
        
        DBUtil.commit(conn);
    } catch (Exception e) {
        DBUtil.rollback(conn);
        throw e;
    }
}
```
