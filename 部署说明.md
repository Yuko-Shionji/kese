# 农村饮水安全监测管理系统 - 部署指南

## 一、环境要求

### 1. JDK 环境
- JDK 版本：1.8 或更高版本
- 配置环境变量：JAVA_HOME, PATH

### 2. 数据库
- MySQL 版本：5.5 或更高版本（推荐免安装版）
- 字符集：utf8

### 3. 开发工具（可选）
- IntelliJ IDEA / Eclipse
- Git

## 二、数据库配置

### 1. 创建数据库
```bash
# 登录 MySQL
mysql -u root -p

# 执行初始化脚本
source database/schema.sql
```

### 2. 验证数据
```sql
USE rural_water_db;

-- 查看表
SHOW TABLES;

-- 查看测试用户
SELECT * FROM users;

-- 查看水厂数据
SELECT * FROM water_plants;

-- 查看检测记录
SELECT COUNT(*) FROM water_quality_records;
```

## 三、项目配置

### 1. 修改数据库连接
编辑文件：`src/com/ruralwater/util/DBUtil.java`

```java
private static final String URL = "jdbc:mysql://localhost:3306/rural_water_db?useSSL=false&characterEncoding=utf8";
private static final String USERNAME = "root";  // 根据实际修改
private static final String PASSWORD = "root";  // 根据实际修改
```

### 2. 导入 MySQL 驱动
将 `mysql-connector-java-5.1.x.jar` 添加到项目依赖

## 四、编译运行

### 方法 1：IDE 直接运行
1. 打开 IDE（IntelliJ IDEA / Eclipse）
2. 导入项目
3. 配置 JDK 和 MySQL 驱动
4. 运行 `MainApp.java`

### 方法 2：命令行编译运行
```bash
# 进入项目目录
cd E:\kese1

# 编译
javac -d bin src/com/ruralwater/**/*.java

# 运行
java -cp bin;lib/mysql-connector-java-5.1.47.jar com.ruralwater.MainApp
```

## 八、常见问题

### 1. 驱动加载失败
确保 mysql-connector-java jar 包在 classpath 中

### 2. 连接数据库失败
检查数据库服务是否启动，用户名密码是否正确

### 3. 中文乱码
确保数据库字符集为 utf8，连接字符串包含 characterEncoding=utf8

## 九、扩展建议

1. **增加图表展示**：使用 JFreeChart 库
2. **导出 Excel**：使用 Apache POI
3. **权限细化**：基于菜单的权限控制
4. **日志记录**：使用 Log4j
5. **连接池**：使用 C3P0 或 Druid
